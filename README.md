# Grafik

GeoFA udstiller ikoner til gratis anvendelse til grafiske produkter (trykt såvel som digitalt).

Til hver facilitet er der piktogrammer i såvel PNG og SVG format og som cirkel, firkant og fritlagt. Filerne har let genkendelige navne i forhold til teksterne i GeoFA-specifikationen.

GeoFA og Naturstyrelsen (NST) er i tæt dialog, så piktogrammer indbyrdes strømlines og gøres fælles. Du kan kende piktogrammer fra NST på den grønne farve samt gennem navngivningen. 

Hvis du finder fejl, mangler, har udviklingsønsker, så skriv til geofa@geodanmark.dk
